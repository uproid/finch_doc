## فینچ CLI
فینچ یک رابط خط فرمان (CLI) برای مدیریت و تعامل با برنامه‌های فینچ شما ارائه می‌دهد. این CLI مجموعه‌ای از دستورات را برای ساده‌سازی کارهای رایج فراهم می‌کند و توسعه، تست و استقرار برنامه‌ها را آسان‌تر می‌سازد.

علاوه بر دستورات ارائه‌شده توسط فینچ، می‌توانید از هر دستور CLI دارت دیگری نیز استفاده کنید. به عنوان مثال، می‌توانید از دستور `dart pub get` برای دریافت وابستگی‌های پروژه خود استفاده کنید. استفاده از CLI فینچ اختیاری است و می‌توانید مستقیماً از CLI دارت نیز بهره ببرید.

## نصب CLI فینچ

```bash
dart pub global activate finch
```

## اجرای CLI فینچ
برای مشاهده لیست دستورات موجود، دستور زیر را اجرا کنید:
```bash
finch -h
```

```bash
دستورات موجود:
1) create:      ساخت پروژه جدید

      --path    مسیر پروژه
      -p

      --name    نام پروژه
      -n

      --docker  استفاده از داکر
      -d
──────────────────────────────
2) get: دریافت پکیج‌های پروژه (معادل dart pub get)
──────────────────────────────
3) runner:      ساخت runner پروژه (معادل dart pub run build_runner build)
──────────────────────────────
4) run: اجرای پروژه (معادل dart run)

      --path    مسیر فایل اپلیکیشن
      -p
──────────────────────────────
5) build:       ساخت پروژه (معادل dart compile exe)

      --appPath مسیر فایل اپلیکیشن
      -a

      --langPath        مسیر زبان‌ها
      -l

      --publicPath      مسیر پوشه public
      -p

      --widgetPath      مسیر ویجت‌ها
      -w

      --envPath مسیر فایل محیط (.env)
      -e

      --output  مسیر خروجی
      -o

      --type    نوع خروجی (zip, exe)
      -t

      --help    نمایش راهنما
      -h
──────────────────────────────
6) test:        اجرای تست واحد پروژه (معادل dart test)

      --reporter        نحوه نمایش نتایج تست
      -r
──────────────────────────────


      --help    نمایش راهنما
      -h

      --version نمایش نسخه فینچ
      -v

      --update  بروزرسانی فینچ
      -u
──────────────────────────────
```

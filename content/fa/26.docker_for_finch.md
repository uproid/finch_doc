# داکر برای فینچ

فینچ برای استفاده با داکر و Nginx ساده‌سازی شده است. می‌توانید با استفاده از فایل `docker-compose.yaml` زیر، برنامه فینچ خود را همراه با Nginx اجرا کنید.

در پوشه example می‌توانید فایل `docker-compose.yaml` را پیدا کنید. می‌توانید از این فایل برای اجرای برنامه نمونه استفاده کنید. همچنین یک `Dockerfile` در ریشه پوشه example وجود دارد که می‌توانید از آن برای ساخت ایمیج برنامه استفاده کنید.

# راهنمای نصب فینچ

به راهنمای نصب فینچ خوش آمدید! این راهنما شما را با مراحل نصب و راه‌اندازی فینچ، یک فریم‌ورک قدرتمند وب برای دارت، آشنا می‌کند. چه توسعه‌دهنده حرفه‌ای باشید یا تازه‌کار، فینچ ابزارهای قدرتمندی برای ساده‌سازی توسعه برنامه‌های سمت سرور ارائه می‌دهد.

## پیش‌نیازها
پیش از شروع، مطمئن شوید که موارد زیر را روی سیستم خود نصب کرده‌اید:

1. **دارت SDK**: فینچ با دارت ساخته شده است، بنابراین باید دارت SDK را نصب کنید. می‌توانید آن را از [وبسایت رسمی دارت](https://dart.dev/get-dart) دریافت کنید.
2. **ویرایشگر متن**: ویرایشگر متن یا IDE دلخواه خود را انتخاب کنید. گزینه‌های محبوب شامل Visual Studio Code، IntelliJ IDEA یا هر ویرایشگری که از دارت پشتیبانی کند است.
3. **ترمینال**: برای اجرای دستورات دارت به یک ترمینال یا خط فرمان نیاز دارید.

## مراحل نصب
برای نصب فینچ، دستور زیر را اجرا کنید:
```bash
dart pub global activate finch
```

## ساخت پروژه جدید
برای ساخت یک پروژه جدید، دستور زیر را اجرا کنید:
```bash
finch create
```

## افزودن فینچ به پروژه موجود
برای افزودن فینچ به پروژه فعلی خود، دستور زیر را اجرا کنید:
```bash
dart pub add finch
```

# انجینکس برای فینچ

فینچ برای استفاده با داکر و Nginx ساده‌سازی شده است. می‌توانید با استفاده از فایل `nginx.conf` زیر، برنامه فینچ خود را همراه با Nginx اجرا کنید.

در پوشه example می‌توانید فایل `nginx.conf` را پیدا کنید. می‌توانید از این فایل برای اجرای برنامه نمونه با nginx استفاده کنید.

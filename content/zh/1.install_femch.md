# Finch 安装指南

欢迎使用 Finch 安装指南！本指南将引导您完成安装和设置 Finch 的步骤，Finch 是一个强大的 Dart Web 框架。无论您是经验丰富的开发人员还是刚刚起步，Finch 都提供了一套强大的工具来简化服务器端 Web 应用程序开发。

## 前提条件
在开始之前，请确保您的系统已安装以下前提条件：

1. **Dart SDK**：Finch 使用 Dart 构建，因此您需要安装 Dart SDK。您可以从[官方 Dart 网站](https://dart.dev/get-dart)下载。
2. **文本编辑器**：选择您喜欢的文本编辑器或 IDE。流行的选项包括 Visual Studio Code、IntelliJ IDEA 或任何其他支持 Dart 的编辑器。
3. **终端**：您需要终端或命令提示符来运行 Dart 命令。

## 安装步骤
使用以下命令安装 Finch：
```bash
dart pub global activate finch
```

## 创建新项目
使用以下命令创建新项目：
```bash
finch create
```

## 将 Finch 添加到现有项目
使用以下命令将 Finch 添加到现有项目：
```bash
dart pub add finch
```

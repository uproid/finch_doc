# 电子邮件发送指南

欢迎使用 Finch 邮件发送指南！本指南将引导您完成在 Finch 应用程序中发送电子邮件的步骤。无论您是经验丰富的开发人员还是刚刚起步，Finch 都提供了一套强大的工具来简化服务器端 Web 应用程序开发。

## 发送邮件
`MailSender` 类用于在 Finch 应用程序中发送电子邮件。它包含发送电子邮件所需的所有属性。以下是使用示例：
```dart
import 'package:finch/src/mail/mail_sender.dart';

bool res = await MailSender.sendEmail(
  from: 'sender@example.com',
  fromName: 'Sender Name',
  host: 'smtp.example.com',
  port: 587,
  to: ['recipient@example.com'],
  subject: 'Test Email',
  html: '<h1>Hello, World!</h1>',
  text: 'Hello, World!',
  username: 'smtp-username',
  password: 'smtp-password',
  ssl: true,
  allowInsecure: false,
);
```

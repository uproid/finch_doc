# Finch 的 Docker

Finch 为 docker 和 Nginx 简化了使用。您可以使用以下 `docker-compose.yaml` 文件与 Nginx 一起运行 Finch 应用程序。

在示例文件夹中，您可以找到 `docker-compose.yaml` 文件。您可以使用此文件运行示例应用程序。您还可以在示例文件夹的根目录中找到 `Dockerfile`。您可以使用此 `Dockerfile` 为示例应用程序构建镜像。

[docker-compose.yaml](https://github.com/uproid/finch/blob/master/example/docker-compose.yaml)

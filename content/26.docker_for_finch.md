# Docker for Finch

Finch is simplified for docker and Nginx. You can use the following `docker-compose.yaml` file to run your Finch application with Nginx.

in example folder you can find a `docker-compose.yaml` file. you can use this file to run the example application. also you can find a `Dockerfile` in the root of the example folder. you can use this `Dockerfile` to build the image for the example application.

[docker-compose.yaml](https://github.com/uproid/finch/blob/master/example/docker-compose.yaml)
